# Biscuit 02 HR⚡ABS

## How to deploy (mobile GitHub method)
1. Download and unzip this folder.
2. Open GitHub app or browser.
3. Create a new public repository: biscuit02-hr-support
4. Upload all files from this folder.
5. Go to Settings → Pages → Branch: main → / (root) → Save
6. Your site will be live shortly!
